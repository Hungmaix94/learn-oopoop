# Learning Resources

This workspace includes comprehensive guides and tutorials for mastering backend development concepts.

## 1. Database Architecture (`base-be`)
*   **Concepts**:
    *   **ACID Properties**: Atomicity, Consistency, Isolation, Durability.
    *   **Normalization**: 1NF, 2NF, 3NF.
    *   **SQL Fundamentals**: CRUD, Joins, Aggregation.
    *   **Indexing & Performance**: When and how to use indexes.
    *   **Security**: Preventing SQL Injection.
*   **Location**: `base-be/` folder.

## 2. NestJS (`nest.js`)
*   **Framework**: NestJS (Node.js).
*   **Paradigm**: Object-Oriented Programming (OOP), Dependency Injection (DI), Decorators.
*   **Key Modules**:
    *   **Controllers**: Handling incoming HTTP requests.
    *   **Providers (Services)**: Business logic encapsulation.
    *   **Modules**: Structural organization.
    *   **DTOs & Validation**: Input validation and data transfer.
*   **Location**: `nest.js/` folder.

## 3. Python (`python`)
*   **Language**: Python 3.
*   **Focus**: Backend development basics.
*   **Key Topics**:
    *   **Syntax**: Variables, loops, conditionals.
    *   **Data Structures**: Lists, Dictionaries.
    *   **Functions**: Arguments, return values, type hinting.
    *   **Error Handling**: Try/Except blocks.
*   **Location**: `python/` folder.
