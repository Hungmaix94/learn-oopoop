# NestJS Zero to Hero - Phần 3: Kết nối Database (TypeORM & Mongoose)

> Một ứng dụng thực tế không thể thiếu Database. NestJS hỗ trợ tuyệt vời cho cả SQL (TypeORM/Prisma) và NoSQL (Mongoose).

---

## 1. Kết nối SQL với TypeORM
TypeORM là ORM phổ biến nhất cho TypeScript. Nó cho phép bạn làm việc với database thông qua các Class (Entities).

```typescript
// Định nghĩa Entity
@Entity()
export class User {
  @PrimaryGeneratedColumn()
  id: number;

  @Column()
  firstName: string;

  @Column({ default: true })
  isActive: boolean;
}
```

**Sử dụng trong Service:**
```typescript
@Injectable()
export class UsersService {
  constructor(
    @InjectRepository(User)
    private usersRepository: Repository<User>,
  ) {}

  findAll(): Promise<User[]> {
    return this.usersRepository.find();
  }
}
```

## 2. Kết nối NoSQL với Mongoose
Nếu bạn chọn MongoDB, NestJS cung cấp module `@nestjs/mongoose`.

```typescript
// Định nghĩa Schema
@Schema()
export class Cat {
  @Prop({ required: true })
  name: string;

  @Prop()
  age: number;
}
export const CatSchema = SchemaFactory.createForClass(Cat);
```

## 3. Configuration (Cấu hình môi trường)
Sử dụng `@nestjs/config` để quản lý các biến môi trường (`.env`) như Database URL, Secret Key.

```typescript
// app.module.ts
ConfigModule.forRoot({
  isGlobal: true, // Dùng được ở mọi nơi mà không cần import lại
})
```

## 4. Authentication với Passport & JWT
NestJS tích hợp sâu với thư viện Passport nổi tiếng để xử lý đăng nhập.

*   **Local Strategy**: Xác thực email/password.
*   **JWT Strategy**: Kiểm tra tính hợp lệ của Token trong các request tiếp theo.
