# NestJS Zero to Hero - Phần 5: Microservices & Real-time (WebSockets)

> Khi ứng dụng của bạn trở nên quá lớn hoặc cần giao tiếp thời gian thực, NestJS cung cấp các giải pháp mạnh mẽ và nhất quán.

## 1. Tại Sao Lại Cần Microservices?

Trong E-commerce, thay vì để một server xử lý tất cả (Monolith), ta tách thành các dịch vụ nhỏ:
*   `Auth Service`
*   `Order Service`
*   `Inventory Service`
*   `Shipping Service`

Lợi ích: Một phần hỏng không làm sập toàn bộ sàn, và có thể scale riêng từng phần.

---

## 2. Microservices trong NestJS

NestJS hỗ trợ nhiều phương thức vận chuyển (Transporters):
*   **TCP**: Mặc định, nhanh, truyền tin trực tiếp.
*   **Redis**: Sử dụng cơ chế Pub/Sub.
*   **RabbitMQ / Kafka**: Dành cho các hệ thống phức tạp, cần hàng đợi tin nhắn (Message Queue) bền vững.

**Ví dụ: Gọi một Microservice khác**

```typescript
@Injectable()
export class OrdersService {
  constructor(
    @Inject('SHIPPING_SERVICE') private client: ClientProxy,
  ) {}

  async createOrder(orderData: any) {
    // Gửi yêu cầu sang Shipping Service mà không cần đợi response block luồng chính
    return this.client.send('order_created', orderData); 
  }
}
```

## 3. Real-time với WebSockets (Gateways)

Dùng cho tính năng: Chat hỗ trợ, Thông báo đơn hàng mới, Cập nhật giá sàn đấu giá real-time.

```typescript
@WebSocketGateway({ cors: true })
export class EventsGateway {
  @WebSocketServer()
  server: Server;

  @SubscribeMessage('events')
  handleMessage(@MessageBody() data: string): string {
    return 'Dữ liệu nhận được: ' + data;
  }

  // Gửi thông báo cho tất cả client đang kết nối
  sendNotification(msg: string) {
    this.server.emit('notification', msg);
  }
}
```

## 4. Hybrid Application

Bạn có thể chạy một ứng dụng vừa là REST API (cho Mobile App) vừa là Microservice (để giao tiếp nội bộ) hoặc vừa là Websocket Server.

```typescript
const app = await NestFactory.create(AppModule);
// Kết nối thêm Microservice vào app hiện tại
app.connectMicroservice<MicroserviceOptions>({
  transport: Transport.TCP,
});

await app.startAllMicroservices();
await app.listen(3000);
```
