# Python Cơ Bản: Cú Pháp Cho Backend Developer

> Hướng dẫn nhanh về các cú pháp cốt lõi của Python, tập trung vào những thứ bạn sẽ dùng hàng ngày khi làm Backend.

---

## 1. Biến & Kiểu Dữ Liệu (Variables & Types)

Python là **Dynamic Typing** nhưng **Strong Typing** (Không tự động ép kiểu bừa bãi như JS).

```python
# Khai báo biến (Cực kỳ đơn giản)
name = "Hung"         # String
age = 30              # Integer
price = 99.99         # Float
is_active = True      # Boolean (Viết hoa chữ đầu)

# Type Hinting (Nên dùng để code sạch như TypeScript)
user_id: int = 123
email: str = "test@example.com"
```

---

## 2. Cấu Trúc Dữ Liệu "Quốc Dân" (Lists & Dicts)

Đây là 2 thứ bạn sẽ dùng 90% thời gian.

### 📜 List (Mảng)
```python
fruits = ["apple", "banana", "cherry"]
fruits.append("orange")  # Thêm vào cuối
print(fruits[0])        # "apple"
```

### 📖 Dictionary (Object/Map)
```python
user = {
    "id": 1,
    "name": "Hung",
    "role": "Admin"
}
print(user["name"])     # "Hung"
```

---

## 3. Điều Khiển Luồng (Control Flow)

**Lưu ý cực quan trọng:** Python dùng **Khoảng trắng (Indentation)** thay vì dấu ngoặc nhọn `{}`.

### If...Else
```python
status = "PAID"

if status == "PAID":
    print("Giao hàng ngay!")
elif status == "PENDING":
    print("Đợi thanh toán...")
else:
    print("Hủy đơn.")
```

### Vòng lặp (For & While)
```python
# Lặp qua danh sách
items = ["Laptop", "Mouse", "Keyboard"]
for item in items:
    print(f"Xử lý: {item}") # f-string (template string)

# Lặp theo dải số
for i in range(5): # 0 đến 4
    print(i)
```

## 4. Hàm (Functions)

Sử dụng từ khóa `def`.

```python
def calculate_total(price: float, quantity: int) -> float:
    """
    Hàm tính tổng tiền đơn hàng.
    """
    return price * quantity

total = calculate_total(50.0, 3)
print(total) # 150.0
```

---

## 5. List Comprehension (Đặc sản Python)

Cách viết vòng lặp "siêu tốc" trên 1 dòng.

```python
prices = [100, 200, 300, 400]

# Lấy các giá > 250
expensive_prices = [p for p in prices if p > 250]
# Kết quả: [300, 400]

# Giảm giá 10% cho tất cả
discounted = [p * 0.9 for p in prices]
```

---

## 6. Xử Lý Lỗi (Try...Except)

Đừng để App sập chỉ vì một lỗi nhỏ.

```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("Lỗi: Không thể chia cho 0!")
except Exception as e:
    print(f"Lỗi không xác định: {e}")
finally:
    print("Luôn thực hiện (ví dụ: đóng kết nối DB)")
```
